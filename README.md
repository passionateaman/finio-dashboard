# Finio – Finance Dashboard

## 🚀 Overview

Finio is a modern and interactive finance dashboard built to help users track their financial activity, visualize spending patterns, and gain useful insights.

## ✨ Features

* Dashboard overview with summary cards (Balance, Income, Expenses)
* Time-based visualization (Balance Trend)
* Category-wise spending breakdown
* Transaction list with filtering and search
* Role-based UI (Viewer & Admin)
* Insights section with key financial metrics
* Responsive design for all screen sizes

## 🛠 Tech Stack

* React + TypeScript
* Vite
* Tailwind CSS
* Zustand (state management)
* Recharts (data visualization)
* Framer Motion (animations)

## 🎯 Key Highlights

* Clean and intuitive UI/UX
* Glassmorphism + glowing effects
* Smooth animations and transitions
* Modular and scalable code structure

## 📦 Setup Instructions

```bash
npm install
npm run dev
```
